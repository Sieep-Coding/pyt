# Development Tools Using Makefile

> [!CAUTION]  
> Using dev tools incorrectly can cause destruction of your data. *Use at your own risk.*

#### Run Local (non-production):
```
make
```
#### Run dev environment (non-production):
```
make dev
```
#### Run *Production Build*:
```
make build
```
#### Cleanup local files:
```
make fresh
```
